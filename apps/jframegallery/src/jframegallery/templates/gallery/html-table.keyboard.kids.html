How to use:

  1. Add the 'keyboardkids' className to the table to activate this plugin
  2. Add data-accesskey attributes on descendants of the table's thead element
  3. Include an alt or title attribute

Demo

Select a row, then press "enter" or "delete" on your keyboard

Those keys are mapped to links
