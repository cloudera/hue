  • I'm the section for Tab 1. Note that my UL has both a .tab_sections class but also a .ccs-clear; that's because the tabs are aligned to the right, so we need to clear that float.

    The tabs are right aligned because their UL has the class .ccs-right; to fix the firefox height bug with floating elements, we also give it the class .clearfix (details on this). Note that if we didn't want the tabs on the right we just wouldn't give it the .ccs-right class, and in that case we wouldn't need the .clearfix or .ccs-clear classes in there either.

    Also note that the tab styling doesn't put this fancy border between the tabs and the content or the padding on the sections. Those styles are up to you.
