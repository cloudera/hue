Hue is an open source graphical user interface to operate and develop applications for performing self-service data analytics: gethue.com.

Thousands of companies and organizations use Hue to open-up and query their data and make smarter decisions. Hue is heavily used by hundreds of Cloudera customers executing millions of queries daily.

2. User Guide

The user guide has instructions about how to use Hue.
