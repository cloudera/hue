Hue v0.9.0, released June 29, 2010

Hue 0.9 is the first open source release of Hue, previously known as Cloudera Desktop. The UI look and feel has greatly improved across different supported browsers (Firefox, Safari, Chrome and IE8). Hue has also made it much easier to write, build, distribute and install an SDK application.
